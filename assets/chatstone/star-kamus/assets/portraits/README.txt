STAR KAMUS — comm portraits
===========================
Drop two images in THIS folder (PNG):

  kamus.png       the pilot (you)
  commander.png   the commander on the mothership

Specs:
- Square, 512x512 or larger (the game shows them at ~76px, object-fit: cover)
- Bust / shoulders-up, face clearly visible
- Dark or space-y background works best (the comm box has its own border)
- Anime style welcome! Kamus' established look: green helmet/bandana, cyan visor.

If a file is missing the game shows a built-in pixel-art placeholder instead.
